# **App Name**: TradeWise

## Core Features:

- Trade Input: Input trade details: amount, predicted return percentage, and win/loss outcome.
- Trade Log: Display a log of all trades with key information (amount, return, current portfolio value).
- Session Statistics: Calculate and display the current portfolio value, win rate, and potential payout.
- Session Reset: Provide a 'reset' function to clear the current session's trade data.
- Smart Trade Amount Suggestion: AI tool helps determine optimal trade amounts based on previous trade history and current portfolio, to maximize profits, conditional on the user-selected risk level. This tool recommends the amount and also alerts the user if taking the amount at their currently history carries a higher probability of bankrupting the portfolio.

## Style Guidelines:

- Primary color: Dark blue (#2E3148) to evoke trust and stability.
- Background color: Very dark blue (#1E202D), almost black, for a modern, focused interface.
- Accent color: Electric green (#00FF7F) to highlight key metrics and calls to action.
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral aesthetic.
- Use minimalist, geometric icons to represent trade outcomes and key functions.
- Use a clean, tabular layout for the trade log to ensure readability.
- Incorporate subtle animations for trade confirmation and portfolio updates.
